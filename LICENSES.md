# LICENSES

В этом проекте используются только собственные ресурсы или ресурсы без внешних зависимостей.

## Файлы проекта
- `index.html` — основной HTML.
- `style.css` — стили проекта.
- `script.js` — логика игры.

## Лицензия
Все файлы проекта предоставляются без ограничений на использование внутри учебного проекта.
Если в дальнейшем будут добавляться сторонние изображения, аудио или шрифты, их лицензии необходимо перечислить в этом файле.
