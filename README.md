# Числовая мозаика

Простой браузерный пазл на чистом JavaScript. Игра предлагает поле 3×3, 4×4 или 5×5, подсчитывает ходы и время, а также сохраняет рекорды в `localStorage`.

## Структура проекта

- `index.html` — основная страница игры.
- `style.css` — стили оформления.
- `script.js` — логика игры.
- `appendices/` — дополнительные материалы и отчеты.
- `appendices/examples/` — примеры исходников и демонстрационные файлы.

## Как запустить

1. Откройте `index.html` в браузере.
2. Выберите сложность.
3. Нажмите на плитку, чтобы переместить её в пустую клетку.

## Заметки

- Игра не использует внешние библиотеки.
- Рекорды хранятся только в локальном хранилище браузера.
- Версия для защиты проекта включает описание процесса и дополнительные пояснения в `appendices/`.
